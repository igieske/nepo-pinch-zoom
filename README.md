# Nepo Pinch Zoom
jQuery plugin for multi-touch pinch zoom and pan of images on mobile devices.

## Installation
Just include nepo-pitch-zoom.jquery.js.

## Usage

### Requirements
You must inlude jQuery before.

### Initialization
```
$('img').nepoPinchZoom();
```

### Options
No options in version 1.0.0.
